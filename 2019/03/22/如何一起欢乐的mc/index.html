如何一起欢乐的mc

首先我们的服务器因为跑的是Java版的所以客户端也当然要是Java版的啦。

Java Runtime Environment

首先第一步肯定是要下载“Java”，这个这里一般都会省略Runtime Environment，所以可能会有弯路（真的会有人百度不到Java嘛？）之后下一步下一步就很容易装好了，Linux也可以一起玩（其实笔者只测试了Windows10和Mac）。

HMCL

这是一个人尽皆知的mc启动器（其实我之前并不是很了解），下载地址，下来应该是对应版本，mac和Linux都是jar文件，装了JRE就能跑了，mac上双击就能跑吧，Windows的话是个exe，在Linux上玩游戏的人应该也不需要我这篇文章了吧。

甚至支持正版登陆（甚至是什么鬼啦？

##Minecraft 整合包！

进入HMCL，点击游戏列表然后是安装整合包（这个东西的表现形式是一个.zip）就行啦！需要进行漫长的等待。你需要的整合包带的mod要和要加入的服务器一样哦～我们的服务器的客户端整合包可以找作者要啦。

Minecraft 多人游戏！！

如果你的计算机运比作者好（作者应该是世界上这玩意最差的人）照着一步步做了那么应该可以打开游戏，你也许会兴奋的退出然后看看能不能分配Java 可用的内存啦这类的。无伤大雅啦。然后点击多人游戏，添加服务器

然后加入游戏就可以啦。记得问问管理员大家都在什么位置玩
